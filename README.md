# Portfólio Mobile

Aplicação mobile desenvolvida com React Native e Expo como atividade prática de desenvolvimento mobile.

## Tecnologias
- React Native
- Expo
- TypeScript
- Expo Router

## Funcionalidades
- Navegação por abas
- Tela sobre/perfil
- Projetos acadêmicos e pessoais
- Modal de detalhes dos projetos
- Links externos
- Jogo da Forca

## Executar o projeto

Instalar dependências:

bash
npm install

Iniciar com Expo:

npx expo start --tunnel

Depois, abrir o Expo Go e escanear o QR Code.
